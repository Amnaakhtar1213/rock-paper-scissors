# Rock-Paper-Scissors Game 🎮

A simple web-based Rock-Paper-Scissors game built with **HTML**, **CSS**, and **JavaScript**.  
Players can choose Rock, Paper, or Scissors, and the computer will randomly select its choice. The game keeps records of scores and displays the result of each round.

---

## Features ✨
-  Interactive buttons for Rock 👊, Paper ✋, and    Scissors ✌.
-  Displays both player and computer choices.
-  Shows the result of each round (Win, Lose, Tie).
-  Keeps track of player and computer scores.
-  Simple and lightweight — runs directly in the browser.

---

##  Technologies

1.  **HTML**
2.  **CSS**
3.  **JAVASCRIPT**
